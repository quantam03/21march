Operators in Javascript :

Arithmatic operators:

+,-,*,/,%,++,-- Ex: I/p: var a=2; var b=3; var sum=a+b; console.log(sum); Sum of two variables using Arithmatic operations O/p: 5

Comparison operators:

===,>==,<==,!== It only returns boolean value ie. true or false Ex: I/p: var a=2; if(a===2) console.log("true"); else console.log("false"); O/p: true

Logical Operators:

&&,||,! Ex: I/p: var a=5; var b=6; if(a===5 && b===6) console.log("true"); else console.log("false"); O/p: true

Assignment operators:

+=,-=,/=,%=, Ex: I/p: var a=2; a+=3; console.log(a); O/p: 5

Ternary Operators:

(condition)? success:failure; Ex: I/p: var a=3; var b=(a===3)? "success": "failure"; console.log(b); O/p: success
